## Changelog : flask-storage (derniers 30 jours)

### Résumé
Les récentes mises à jour de flask-storage incluent l'ajout de la configuration optionnelle du nom du bucket S3, facilitant l'utilisation du backend S3. Une nouvelle version de développement a également été préparée pour le prochain cycle.

### Évolutions fonctionnelles
- Ajout de la possibilité de configurer le nom du bucket S3 via une option de configuration. (#16)

### Évolutions techniques
- Préparation de la version 1.4.4.dev pour le prochain cycle de développement.
- Bump de version à 1.4.3.
- Mise à jour du fichier CHANGELOG.
