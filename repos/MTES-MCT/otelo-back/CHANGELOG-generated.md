## Changelog : otelo-back (derniers 30 jours)

### Résumé
Les dernières mises à jour d'otelo-back se concentrent sur l'amélioration de la page de résultats, la correction de bugs dans les calculs et l'ajout de nouvelles données démographiques par territoire. Une fonctionnalité permettant d'importer la base de données en local a également été ajoutée pour faciliter le développement et les tests.

### Évolutions fonctionnelles
- Refonte complète de la page de résultats (v2) (#79)
- Ajout des données démographiques par territoire (#80)
- Correction d'un bug dans le calcul des totaux de logements (#80)
- Correction d'un problème de garde pour les résultats (#1643db4)

### Évolutions techniques
- Ajout d'une commande pour importer la base de données en local, facilitant le développement et les tests (#80)

### Autres changements
- Aucun changement significatif à signaler.
