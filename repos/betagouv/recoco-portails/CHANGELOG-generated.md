## Changelog : recoco-portails (derniers 30 jours)

### Résumé
Ce changelog décrit les améliorations apportées aux portails de recoconseil.fr au cours des dernières semaines. Les modifications incluent des mises à jour de contenu pour le portail SOS Ponts, des améliorations d'accessibilité pour les liens externes, des optimisations de performance via la compression d'images et une unification des fichiers CSS pour une meilleure maintenance.

### Évolutions fonctionnelles
- Mise à jour de la page d'accueil du portail SOS Ponts (#193, #90444b7, #584eefd).
- Amélioration de l'accessibilité des liens ouvrant une nouvelle fenêtre grâce à un changement de classe (#191, #a4ffdb9).
- Ajout d'un lien vierge et amélioration de l'accessibilité de tous les liens externes, qui s'ouvrent désormais dans un nouvel onglet (#190, #43c7521).
- Mise à jour du texte de la page de confidentialité PUI-ASU v2 (#188, #9304959).
- Correction du style du lien de fichier UV/GNIUS/PUI (#183, #0410b21).

### Évolutions techniques
- Optimisation des images pour améliorer les performances (#184, #8aab7c9).
- Unification des fichiers CSS pour faciliter la maintenance et améliorer la cohérence du code (#187, #f0ba463).
- Mise en place de Vite pour servir les assets (#183, #9e7d90b).
- Correction de l'appel des fichiers SCSS pour la page de confidentialité (#187, #9304959).
- Correction de l'intégration de la branche `main` dans plusieurs branches de fonctionnalités (#193, #191, #185, #189, #188, #184, #190, #187).
