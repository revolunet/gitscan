## Changelog : oauth2-deploy-demo (derniers 30 jours)

### Résumé
Ce projet a reçu une petite mise à jour de documentation pour clarifier l'utilisation des attributs personnalisés (custom claims) lors de la configuration de l'authentification OAuth2. Cette amélioration permet aux utilisateurs de mieux comprendre comment adapter l'authentification à leurs besoins spécifiques.

### Évolutions techniques
- Ajout d'une note dans la documentation concernant l'utilisation des attributs personnalisés (custom claims) pour une configuration plus flexible de l'authentification OAuth2. (#I00000000000000000000)
