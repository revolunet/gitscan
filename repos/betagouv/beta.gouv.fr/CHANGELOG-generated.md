## Changelog : beta.gouv.fr (derniers 30 jours)

### Résumé
Ce changelog résume les récentes mises à jour de beta.gouv.fr. Les améliorations concernent principalement l'API avec l'ajout d'informations sur les technologies utilisées par les startups, ainsi que l'exposition de l'URL d'impact des projets. Des mises à jour de la base de données ont également été effectuées, ainsi que des corrections mineures sur le site web.

### Évolutions fonctionnelles
- L'API expose désormais les technologies utilisées par les startups (#21434).
- L'API expose maintenant le champ `impact_url` pour chaque projet (#21422).
- Le lien vers le blog a été remplacé par un lien vers les standards d'accessibilité (#21416).
- La liste des startups ne comprend plus les incubateurs "sgmas" hérités (#21427).

### Évolutions techniques
- Mises à jour régulières de la base de données (#21435, #21432, #21431, #21430, #21428, #21426, #21424, #21423, #21421, #21420, #21419, #21414, #21412, #21411, #21409).
- Ajout de la documentation de l'API (#21433).

### Autres changements
- Bilan du schéma pluriannuel d'accessibilité 2025 (#21404).
