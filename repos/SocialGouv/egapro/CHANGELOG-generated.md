## Changelog : egapro (derniers 30 jours)

### Résumé
Les dernières mises à jour d'EgaPro permettent aux entreprises de réaliser leur déclaration d'index égalité pour l'année 2026. Des corrections de bugs ont également été apportées pour améliorer la diffusion des informations, notamment le code NAF et les données REPEQ. De plus, l'application utilise désormais l'API annuaire entreprise pour une meilleure gestion des données.

### Évolutions fonctionnelles
- Ouverture des déclarations d'index égalité pour l'année 2026 (#2731)
- Utilisation de l'API annuaire entreprise pour la gestion des données de l'entreprise (#2728)
- Correction d'un bug empêchant la diffusion du code NAF.
- Correction d'un bug empêchant la diffusion des données REPEQ (#2739).
- Correction d'une erreur non diffusible.

### Évolutions techniques
- Aucune évolution technique significative à signaler durant cette période.

### Autres changements
- Publication des versions 3.15.1 et 3.15.0.
